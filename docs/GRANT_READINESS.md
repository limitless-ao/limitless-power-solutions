## 2. Grant-readiness checklist (mapped to ZE-Gen-style criteria)

The R&D program (Integrated Renewable Energy System) is intentionally designed to meet zero-emission generator funding criteria.

- [x] **Direct fossil-fuel generator replacement** — explicit objective of the program
- [x] **Safe, flexible, affordable, reliable, low-carbon** — all four are stated design principles
- [x] **Multi-sector applicability** — commercial, domestic, public sector, humanitarian
- [x] **Hybrid AC + DC renewable generation** — designed in
- [x] **Grid-charge backup** — included as a fallback charging path
- [x] **Energy storage capability** — integrated battery system
- [x] **Scalability** — modular capacity from small to enterprise
- [x] **Local supply-chain utilization** — installation, maintenance and (over time) manufacturing
- [x] **End-of-life circularity plan** — reuse / recycle / repurpose pathway documented
- [x] **Pilot programs proposed** — multiple sectors and use cases
- [x] **Local capacity building** — technician training programs
- [x] **Continuous improvement** — iterative methodology

## 4. Positioning principles for applications

When applying, make sure each submission emphasizes:

1. **Specificity over ambition.** A pilot in one named community beats "transforming Africa".
2. **Measurable impact.** Every claim has a unit — kWh, ₦, hours of power, tonnes of CO₂.
3. **Local credibility.** Nigerian-registered, Nigerian-led, Nigerian crews. This is an asset.
4. **Realistic timeline.** Funders distrust projects that promise full deployment in 6 months.
5. **Co-funding & in-kind.** Show how the grant is one piece of a larger funding mix.
6. **Honest risks.** A risk register that names regulatory, technical, financial, social risks (and mitigations) is more credible than one that claims none exist.

## 5. Website's role in funding readiness

The website is a credibility instrument. For each funder visit, it must show:

- A real registered business (About page)
- A coherent commercial practice that funds operations (Services + Projects)
- A research / community arm with a clear methodology (Microgrid + Research)
- A measurable impact framework (Impact + SDG alignment)
- A funder-specific entry point (Investors page + data-room request)

Keep the **Investors** page up to date — that's where due-diligence visitors will land.
